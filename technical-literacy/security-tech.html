Network Security

Firewalls: Provide visibility and control at the network boundary.

  • Next-Gen Firewalls (NGFW): Go beyond port/protocol filtering to include deep packet inspection and application awareness.
  • Web Application Firewalls (WAF): Specifically designed to protect web applications from common attacks like SQLi and XSS.
  • IDS/IPS: Intrusion Detection/Prevention Systems monitor for malicious activity or policy violations.
  • Proxies: Act as intermediaries for requests, providing anonymity and content filtering.

Endpoint Security

EDR/XDR: Modern endpoint protection that focuses on behavioral detection and response.

  • Antivirus (AV): Traditionally signature-based, focused on known malicious files.
  • Endpoint Detection & Response (EDR): Continuously monitors endpoint events (processes, network connections, file changes) for suspicious behavior.
  • XDR: Extended Detection and Response, integrating data from multiple sources (network, cloud, email) for a holistic view.
  • Host-based Firewalls: Control traffic at the individual device level (e.g., Windows Firewall).

Log Management

SIEM: Security Information and Event Management systems aggregate and analyze security data.

  • Log Aggregation: Collecting logs from various sources (firewalls, endpoints, servers) into a single location.
  • Centralized Logging: Essential for correlation and incident response across the enterprise.
  • Decentralized Logging: Logs reside on individual systems, making holistic analysis difficult.
  • SIEM Goals: Real-time monitoring, correlation of events, and long-term storage for forensic analysis.
