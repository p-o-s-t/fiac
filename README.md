# FIAC - Future Intelligence Analyst Course
Welcome to the Future Intelligence Analyst Course (FIAC), an interactive, static training platform designed for aspiring Cyber Threat Intelligence (CTI) practitioners.  This project is also hosted on a [GitHub Page](https://p-o-s-t.github.io/cti-training-page/index.html).

## Overview
FIAC provides a gamified learning experience covering the fundamental knowledge and practical skills required for the All-Source Analyst (DCWF 111) work role. The platform is designed to be simplistic, human-readable, and easily accessible.

## Features
- Dashboard Centric: A single-viewport interface for navigating lessons and challenges.
- Knowledge Modules: Lessons ranging from Intelligence Fundamentals to Data Management, each featuring a no-scroll slide deck and a randomized Knowledge Check.
- Interactive Challenges: Practical exercises including SSL/TLS Infrastructure Pivoting, MITRE ATT&CK Mapping, and Diamond Model Reconstruction.
- Progress Tracking: Session-based progress saved to localStorage, with visual indicators (checkmarks) on the dashboard.
- DCWF Alignment: Every module is mapped to specific Knowledge, Skills, and Abilities (KSAs) from the Defense Cyber Workforce Framework.
- Resources Library: A categorized collection of frameworks, tools, and research links.

## Technical Details
- Frontend: Vanilla HTML5, CSS3 (using CSS Variables), and ES6+ JavaScript.
- AI Assisted: This project was developed with the assistance of Gemini, which helped in structuring the curriculum, generating interactive challenges, and ensuring DCWF alignment.
- No Frameworks: Built without heavy frameworks to ensure the code itself is an educational resource.
- Mobile Responsive: Layouts adjust for different screen sizes while maintaining the no-scroll constraint for lessons.
